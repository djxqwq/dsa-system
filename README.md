# DSA System

驾校管理系统 - A Spring Boot application for driving school management.

## 技术栈

- Spring Boot 3.5.11
- Java 17
- Maven
- MySQL

## 本地运行

1. 确保安装 JDK 17 和 Maven。
2. 配置 MySQL 数据库。
3. 运行后端： 
```bash   
   mvn spring-boot:run
```
   4. 启动前端：
```bash
   cd frontend
   npm install #首次运行需要安装依赖
   npm run dev
 ```

## 构建

```bash
   mvn clean install
```
